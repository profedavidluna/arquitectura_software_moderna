# Architecture Diagrams

This directory contains all architecture diagrams for the Software Architecture Course.

## Diagram Types

### C4 Model Diagrams
- **Level 1 - System Context**: High-level system overview
- **Level 2 - Container**: Major containers and their interactions
- **Level 3 - Component**: Internal components of each service
- **Level 4 - Code**: Code-level details (classes, methods)

### UML Diagrams
- **Class Diagrams**: Entity relationships and class structures
- **Sequence Diagrams**: Interaction sequences for key workflows
- **State Diagrams**: State machines for order status, payment status, etc.
- **Use Case Diagrams**: User interactions and system functionality

### Database Diagrams
- **ER Diagrams**: Entity-relationship diagrams for each database
- **Schema Diagrams**: Database schema and relationships

### Other Diagrams
- **Component Diagrams**: Microservice components and dependencies
- **Deployment Diagrams**: Infrastructure and deployment architecture
- **Data Flow Diagrams**: Data flow between services

## Tools Used

- **Draw.io**: For creating diagrams (export as PNG, SVG, PDF)
- **PlantUML**: For UML diagrams (text-based)
- **Mermaid**: For flowcharts and diagrams (markdown-based)
