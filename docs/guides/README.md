# Guides and Tutorials

This directory contains guides and tutorials for the Software Architecture Course.

## Contents

- **Getting Started**: Quick start guide for setting up the development environment
- **Local Development**: Guide for running services locally with Docker Compose
- **API Usage**: Guide for using the microservices APIs
- **Testing**: Guide for running tests and achieving code coverage
- **Deployment**: Guide for deploying to production
- **Troubleshooting**: Common issues and solutions
- **Best Practices**: Coding standards and best practices
- **Design Patterns**: Implementation guides for design patterns
- **SOLID Principles**: Explanation and examples of SOLID principles
