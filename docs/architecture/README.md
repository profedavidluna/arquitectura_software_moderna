# Architecture Documentation

This directory contains all architecture-related documentation for the Software Architecture Course.

## Contents

- **C4 Model Diagrams**: System context, container, component, and code-level diagrams
- **Component Diagrams**: Detailed component architecture for each microservice
- **Sequence Diagrams**: Key workflow sequences (order creation, payment processing, etc.)
- **UML Diagrams**: Class diagrams, state machines, use cases
- **ER Diagrams**: Database entity-relationship diagrams
- **Architecture Patterns**: Documentation of implemented patterns (Saga, Circuit Breaker, etc.)
